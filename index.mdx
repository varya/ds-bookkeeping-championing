## What is it?

This is a project template which lets you create your beautiful [Shower](https://shwr.me/) presentations with markdown syntax and custom React components.

The following slides demonstrate rendered HTML. Check [index.mdx](https://github.com/varya/shower-mdx/blob/master/index.mdx) to see how it looks in markdown.

[Project repository](https://github.com/varya/shower-mdx)

*Feedback and contributions are warmly welcomed!*


## Shower key features

1. Built on HTML, CSS and JavaScript
1. Works in all modern browsers
1. Themes are separated from engine
1. Fully keyboard accessible
1. Printable to PDF

## Code samples 1: simple code block with markdown syntax

```html
<!DOCTYPE html>
<html lang="en">
<head> <!--Comment-->
    <title>Shower</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="screen.css">
</head>
```
